Iterative Alignment (T-Coffee) 	[latex]\begin{enumerate}    \item align sequences to initial MSA $A$    \item for each sequence $s$        \begin{itemize}            \item remove $s$ from MSA $A$            \item remove gap-only columns from MSA            \item realign $s$ to MSA            \item keep new MSA if overall score improves        \end{itemize}    \item repeat (2) until convergence\end{enumerate}[/latex]
Alternatives to Progressive Alignment (T-Coffee)  	[latex]\begin{itemize}    \item iterative alignment: try to correct by removing sequences andrealigning them correctly afterwards    \item consistency-based: try to avoid to introduce errors by keepingalignments consistent with rest\end{itemize}[/latex]
How to split and realign $k$ sequences? Options.. (T-Coffee)  	[latex]\begin{enumerate}    \item Round Robin: realign sequences in order of input to multiple alignment\item general single-type partitioning $\Rightarrow$ k-1 partitions possible\item double-type partitioning: choose up to $2$ sequences for realign\\$\Rightarrow \frac{k(k-1)}{2} = \binom{k}{2}$ partitions possible \item choose any subset for realign\\$\Rightarrow 2^k - 2$ possibilities\item feasible solution: tree-dependeent iterative method\end{enumerate}[/latex]
Iterative Alignment: Deterministic vs Stochastic 	[latex]\begin{itemize}    \item deterministic (see different card): round robin, double-type partitioning, realigningarbitrary subsets, tree-dependent iterative method    \item stochastic: \begin{enumerate}    \item simulated annealing\begin{itemize}    \item randomly change alignment    \item accept according to acceptance function\end{itemize}    \item genetic algorithm\end{enumerate}\end{itemize}[/latex]
