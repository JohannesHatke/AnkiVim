Nondeterministic planning - Motivation  	world not predictable, examples: AI robotics (robot movements), games (otherplayers outside of own control)
Nondeterministic Planning - General Idea  	Deterministic planning:only changes taking place induced by us and we can exactly predict results ofour actionsNondeterministic planning:Other agents + processes formalized as nondeterminism=> Implications:1. Future state of world cannot be predicted2. Cannot reliably plan ahead: no single operator sequence achieves goals3. In some cases: impossible to achieve goal with certainty no matter whichoutcomes actions have, but only under certain fairness assumptions
Definition - Nondeterministic transition system  	[latex]A Nondeterministic transition system $ \mathcal{T}= \langle S, L, T, s_o, S_{*} \rangle$ with\begin{itemize}\item $S$ states\item $L$ set of labels\item $T \subseteq S \times L \times S$ transition relation\item $s_0 \in S$ initial state\item $S_{*} \subseteq S$ set of goal states\end{itemize}Note: $T \subseteq S \times L \times S$ allows nondeterministic operators withmore than a single possible outcome[/latex]
Definition - Nondeterministic operators  	[latex]Let $V$ set of finite-domain state vars. A nondeterministic operator is a pair $o = \langle \xi, E \rangle$ with\begin{enumerate}\item $\xi$ conjunction of atoms over $V$ (precondition)\item $E = \{e_1, \dots, e_n \}$ finite set of possible effects of $o$, each$e_i$ being conjunction of atomic finite-domain effects over $V$\end{enumerate}[/latex]
Definition - Nondeterministic operator application  	[latex]Let $o = \langle \xi, E \rangle$ nondeterministic operator, $s$ state. \\$o$ is applicable in $s$ iff $s \models \xi$ and change set of each effect $e\in E$ is consistent.\\ \\Application of $o$ in $s$ leads to one of the states in the set:\\ $app_{o}(s) := \{app_{\langle \xi, e \rangle}(s) | e \in E\}$nondeterministically[/latex]
Definition - nondeterministic planning task  	[latex]A nondeterministc task is $\Pi = \langle V,I,O,\gamma \rangle$ with\begin{enumerate}\item $V$ finite set of finite-domain variables\item $I$ initial state over $V$\item $O$ finite set of nondeterministc operators over $V$\item $\gamma$ conjunctions of atoms over $V$ describing goal states\end{enumerate}[/latex]
Definition - transition system induced by nondeterministic planning task 	[latex]Every nondeterministic task $\Pi = \langle V, I, O,\gamma \rangle$ inducescorresponding nondeterministic transition system $ \mathcal{T}(\Pi) = \langleS,L,T,s_0, S_{*} \rangle$ with\begin{enumerate}\item $S$ set of states\item $L$ set of operators\item $T = \{\langle s,o, s' \rangle | s \in S, o$ applicable in $s, s' \inapp_{o}(s)\}$\item $s_0 = I$\item $S_{*} = \{s \in S | s \models \gamma \}$\end{enumerate}[/latex]
Definition - Strategy (nondeterministic planning)  	[latex]Let $\Pi = \langle V,I,O,\gamma \rangle$ nondeterministic task with states $S$and goal states $S_{*}$.\\ \\A strategy for $\Pi$ is function $\pi:S_{\pi} \to O$ for some subset $S_{\pi}\subseteq S$ s.t. for all states $s \in S_{\pi}$ action $\pi(s)$ applicablein $s$.\\ \\Set of states reachable in $ \mathcal{T}(\Pi)$ starting in state $s$ andfollowing $\pi$ denoted by $S_{\pi}(s)$[/latex]
Nondeterministic plans - Definition  	[latex]Let $\Pi = \langle V,I,O,\gamma \rangle$ Nondeterministic task with states $S$,goal states $S_{*}$ and $\pi$ strategy for $\Pi$.\\Then $\pi$ is called\begin{itemize}\item weak iff $S_{\pi}(s_0) \cap S_* \neq \emptyset$\item closed iff $S_{\pi}(s_0) \subseteq S_{\pi} \cup S_*$\item proper iff $S_{\pi}(s') \cap S_* \neq \emtpyset $ for all $s'\inS_{\pi}(s_0)$\item acyclic iff exists no state $s' \in S_{\pi}(s_0)$ s.t. $s'$ reachablefrom $s'$ following $\pi$ in strictly positive number of steps.\end{itemize}[/latex]
Nondeterministic plans - remarks  	-> Strategies in Nondeterministic planning = applicable operator sequences indeterministic planning-> in deterministic planning, plan = applicable operator sequence resulting ingoal state-> in nondeterministic planning, different notions of resulting in goal stateexist
Nondeterministic planning - Definition of notions of "ending in goal state"  	[latex]Let $\Pi = \langle V,I,O,\gamma \rangle$ Nondeterministic task with states $S$and goal states $S_*$\begin{itemize}\item strategy called weak plan iff it is weak\item strategy called strong cyclic plan iff it is closed and proper\item strong cyclic plan called strong plan iff it is acyclic\end{itemize}[/latex]
