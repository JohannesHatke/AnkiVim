Labeled state transition systems: Definition 	[latex]A labeled state transition system (LSTS) is $\mathcal{LSTS} = \langle \Sigma,Lab, Edge, Init \rangle$ with\begin{itemize}\item a (probably infinite) state set $\Sigma$\item a label set $Lab$\item a transition relation $Edge \subseteq \Sigma \times Lab \times \Sigma$\item a non-empty set of initial states $Init \subseteq \Sigma$\end{itemize}[/latex]
Semantics of labeled state transition systems  	[latex]Operational semantics:\begin{equation*}\frac{(\sigma, a, \sigma') \in Edge}{\sigma \stackrel{a}{\to}\sigma'}\end{equation*}\begin{itemize}\item system run (execution):\begin{equation*}\sigma_0 \stackrel{a_0} \sigma_1 \stackrel{a_1} \sigma_2 \dots$\end{equation*}where $\sigma_0 \in Init$\item a state is called reachable iff there is a run leading to it\end{itemize}[/latex]
Parallel composition: Intuition 	[latex]larger or more complex systems are often modeled compositionally\begin{enumerate}\item global system is given by parallel composition of components\item component-local (non-synchronizing) transitions have labelsbelonging only to a single components label set.\\ They are executed in an interleaved manner\item synchronizing transitions of the components agree on the label. They areexecuted synchronously.\end{enumerate}[/latex]
